How to play!

Think you have what it takes to challenge your color sorting and coordination skills?. hhere are the rules on how to play the game:

  • To select a nut: Click on a nut to select it.
  • To move the selected nut: Click on the desired bolt to drop it.
  • Stack smartly: Nuts can only be placed on another nut of the same color. If the bolt is empty, any nut can go there!
  • Need more space? You can add one extra bolt, but it only holds one nut at a time!
  • Made a mistake? No worries! You can undo up to 2 moves per level. Click undo button "xxx"
  • Limited moves: You have a limited number of moves to complete each level. If you run out, you lose!
  • Got stuck? Click reset button "xxx" to try again!
  • Win the game: Stack all the colors correctly and become the stacking champion!

Let's have fun and stack those colors like a pro!
