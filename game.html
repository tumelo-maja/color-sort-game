Help

How to play

Unscrew: Click on a nut to unscrew it.

Move nut: Click on the desired bolt to move it.

Stack smartly: Nuts can only be placed on another nut of the same color. If the bolt is empty, any nut can go there!

Limited moves: If you run out if moves before completing the sorting, you lose!

undo Undo last move: Click to reverse the last move - one use per move

add Extra bolt: Click to add activate extra bolt

settings_backup_restore Reset level Click to restart the level

splitscreen_vertical_add New game: Click to load new game

Win the game: Stack all the colors correctly and become the stacking champion!

Score calculation

You get:

10 points for each completed bolt

1 point for every nut placed

10 bonus points if you do not use the extra bolt

5 bonus points for every move remaining at the end

Keyboard shortcuts

Ctrl+Z: Undo last move undo

Shift+X: Extra bolt add

Shift+R: Restart level settings_backup_restore

Shift+N: Load new game splitscreen_vertical_add

Shift+M: Toggle sound
